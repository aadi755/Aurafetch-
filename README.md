# 🌌 AuraFetch ✨

A beautiful, all-in-one system info tool — better than Neofetch or Fastfetch.

---

## 🔧 Features

- 🖥️ OS, Kernel, Hostname  
- ⏳ Uptime  
- ⚙️ CPU Info & Temperature (Linux only)  
- 🎮 GPU Info (via `lspci`)  
- 🧠 RAM Usage  
- 💾 Disk Usage  
- 🌐 Local & Public IP  
- 💻 Shell & Terminal Info  
- 📺 Screen Resolution  
- 📦 Package Count (Debian, Arch, RPM supported)
